This is a demo of the new Angular Router in Angular 1.x applications.

The router is in early stages still and a couple of the things things planned for this demo are still incomplete (linking to child routes throws an undefined for some reason, including the demo app from the official repo).

This demo will add to the great routing talks already given by Rob Eisenberg and Brian Ford.

I've written on ngRoute and UI-Router before, and I'd like to make this demo more code driven in a real-world application.

Main Points:

  • Using the new Angular router in 1.x applications
  • Why component based routing is neat (reusable, nestable, sibling-able?)
  • Building a quick application real-time helps explain components instead of just looking over an existing app)
